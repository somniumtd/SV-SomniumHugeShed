*Huge Shed for Stardew Valley*

Author: Somnium
Version: 1.0.0
Description: Adds larger room and cellar to shed

Requirements:
 - SMAPI (https://smapi.io/)
 - Content Patcher (https://www.nexusmods.com/stardewvalley/mods/1915)

Installation:
 - Download the latest version of all the above mods and unzip them into your game folder (Stardew Valley/Mods)
 - Download and unzip this mod into Stardew Valley/Mods
 - Run the game through SMAPI
